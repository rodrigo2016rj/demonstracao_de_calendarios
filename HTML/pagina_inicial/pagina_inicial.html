Página Inicial

Selecione qual calendário deseja ver o resultado pelo menu do cabeçalho do sistema. Este sistema demonstra o resultado de quatro formas de fazer calendário em campos de selecionar data:

No segundo link, JavaScript, o sistema mostra o resultado do calendário feito apenas com JavaScript, sem o uso de quaisquer frameworks para JavaScript. Por isso chamei de JavaScript puro.

No terceiro link, JQuery, o sistema mostra o resultado do calendário feito com JQuery.

No quarto link, React, o sistema mostra o resultado do calendário feito com React.

No quinto link, Vue, o sistema mostra o resultado do calendário feito com Vue.
